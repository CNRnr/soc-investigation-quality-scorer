
# Attack Coverage

This project dataset was generated by simulating multiple ATT&CK techniques in a controlled VMware cyber range using Atomic Red Team. Telemetry was collected through Splunk SIEM and used to build the SOC investigation dataset.

The following techniques are currently covered.

| MITRE Technique | Technique Name | ATT&CK Stage | Primary Telemetry | Example Command |
|---|---|---|---|---|
| T1546.013 | PowerShell Profile Modification | Persistence | PowerShell ScriptBlock (4104), Process Creation (4688) | Add-Content $PROFILE |
| T1518 | Software Discovery | Discovery | Process Creation (4688) | reg query HKLM |
| T1140 | Deobfuscate / Decode Files | Defense Evasion | Process Creation (4688) | certutil -decode |
| T1124 | System Time Discovery | Discovery | PowerShell ScriptBlock, Process Creation | Get-Date / wmic os get LocalDateTime |
| T1083 | File and Directory Discovery | Discovery | PowerShell ScriptBlock | Get-ChildItem -Recurse |
| T1082 | Environment Variable Discovery | Discovery | PowerShell ScriptBlock | Get-ChildItem Env: |
| T1069.001 | Permission Groups Discovery | Discovery | PowerShell ScriptBlock, Process Creation | net localgroup |
| T1057 | Process Discovery | Discovery | Process Creation (4688) | tasklist /svc |
| T1049 | Network Connection Discovery | Discovery | PowerShell ScriptBlock, Process Creation | netstat -ano |
| T1012 | Registry Query | Discovery | PowerShell ScriptBlock | Get-ItemProperty HKLM |

## Dataset Characteristics

- Techniques: **10**
- Scenarios per technique: **10**
- Investigation variants: **Good / Medium / Poor**
- Total dataset size: **300 samples**

Each dataset entry contains:
- raw security log
- analyst investigation text
- investigation quality scores
